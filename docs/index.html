MDDIFF

Markdown Diff Editor for macOS

Download for Mac (Apple Silicon) View on GitHub

Features

+

Always-on Diff

Original (read-only) vs. modified (editable) side by side. Lines unique to each side highlighted in green.

📍

Cursor Sync

Moving the cursor on the right highlights the corresponding original line on the left. Line-synced scrolling.

👁

Markdown Preview

Toggle rendered preview with diff markers preserved. Tables, code blocks, headings all rendered.

📑

Table of Contents

Navigate headings via dropdown. Click to jump to the section in editor or preview.

▲▼

Delta Navigation

Jump between changes with arrow buttons. Both sides scroll to the corresponding position.

💾

History & Save

Last 5 comparisons stored. Save modified file. Dark/light theme. Drag & drop support.
