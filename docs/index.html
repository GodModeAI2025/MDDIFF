MDDIFF

Side-by-side Markdown Diff Editor for macOS

Download for Mac (Apple Silicon) View on GitHub

Features

📄⇄📄

Split View

Open two Markdown files side by side. Edit both simultaneously with full editor support.

+ −

Git-style Diff

Line-by-line comparison with color-coded additions and deletions, just like git diff.

👁

Diff Preview

View diffs as rendered Markdown with highlighted changes, not just raw source.

📝

Edit & Preview

Toggle each panel between edit mode and rendered Markdown preview independently.

🔃

Synced Scroll

Both panels scroll in sync — in editor, preview, and diff view.

🎯

MD Only

Purpose-built for Markdown. File dialogs filter for .md files. Drag & drop supported.
